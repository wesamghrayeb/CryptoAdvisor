# AI Usage Disclosure

AI tools such as Cursor and ChatGPT were used during this assignment as development assistants.

I used AI mainly to help with planning the project structure, improving code organization, reviewing implementation ideas, debugging small issues, and writing clearer documentation.

The main architecture decisions, environment setup, API configuration, database setup, testing, deployment preparation, and final code review were done manually.

I reviewed every AI-assisted change before keeping it in the project. I made sure the code stayed simple, readable, secure, and suitable for a developer who needs to maintain a real product independently.

No secrets, API keys, database credentials, or environment values were committed to the repository.

AI helped me work faster, but the final implementation reflects my own understanding, decisions, testing, and ownership of the project.
